# only-bt-scan

A Rust project for Bluetooth scanning.

## Getting Started

### Prerequisites

- Rust 1.56 or later
- Cargo (comes with Rust)

### Installation

1. Clone or download the project
2. Navigate to the project directory
3. Build the project:

```bash
cargo build
```

### Usage

Run the application:

```bash
cargo run
```

## Development

- Source code is in the `src/` directory
- Edit `src/main.rs` to modify the application
- Build with `cargo build --release` for optimized builds
- Run tests with `cargo test`
